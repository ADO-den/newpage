How it works

  1. Select a package and any add-ons you want.
  2. Tap "Order via WhatsApp" to send your order details (we'll confirm availability and delivery).
  3. Provide recipient name, delivery address, preferred delivery date, and a short message for the card.
  4. We prepare, wrap, and dispatch. You receive tracking and delivery confirmation.

All packages include premium wrapping. Returns and exchanges depend on package contents — contact support for details.
